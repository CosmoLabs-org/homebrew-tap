# CosmoLabs Homebrew Tap

## Usage

```bash
brew tap cosmolabs-org/tap
brew install supermac
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| supermac | macOS system management CLI — 12 modules, 127 commands |

## SuperMac

A modular CLI for macOS system management. Network, display, audio, dock, dev tools, and more.

```bash
mac doctor          # System health check
mac network ip      # Show network info
mac wifi status     # WiFi status
mac --version       # Version info
```
